# 产品详细设计文档生成 Skill

专为产品经理设计的文档生成工具，符合中文学术格式规范。

## 快速开始

```
请帮我为[产品名称]生成产品详细设计文档
```

## 核心特性

- ✅ 符合中文学术格式（A4、宋体小四、1.5倍行距）
- ✅ 九大核心模块
- ✅ 删除技术细节，专注产品设计

## 文档清单

- `template.md` - 文档模板
- `README.md` - 本说明文件

## 格式规范

- **纸张**：A4，边距左右3.18cm，上下2.54cm
- **字体**：正文宋体小四，标题黑体
- **层级**：一、 → （一） → 1. → （1） → ①
